# Chocolate Sales Analysis Project

## Overview

This project analyzes a Chocolate Sales dataset using Python, Pandas, Matplotlib, and SQL.

## Task 1: Data Cleaning

* Removed duplicate records
* Checked and handled missing values
* Standardized data
* Created cleaned dataset

## Task 2: Exploratory Data Analysis

* Calculated total sales
* Identified top-selling products
* Analyzed sales distribution
* Created visualizations:

  * Bar Chart
  * Pie Chart
  * Histogram

## SQL Analysis

* Total Sales Query
* Product-wise Sales Query
* Revenue Analysis Queries

## Business Insights

* Top-selling products generated the highest revenue.
* Product sales varied across categories.
* Data analysis revealed business trends.
* High-performing products can be prioritized for future sales growth.
* Sales distribution analysis helped identify differences in transaction values.
* Data-driven insights can support better business decision-making.
